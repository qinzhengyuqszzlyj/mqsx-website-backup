5月9日下午，初中部八年级开展英语学科活动暨Wonder整本书阅读 “乐“读 “乐”精彩英语故事表演比赛。为了提升学生英语学习的兴趣和自信心，拓宽学生的英语语用渠道，为学生提供一个展示自我的平台，八年级英语组结合书本《Wonder》展开了本次活动。此次英语学科活动分为英语读书笔记、读书小报和英语课本剧表演。
前期，八年级同学们利用周末的时间阅读了《Wonder》这一本书，并用Reading Log记录了读书笔记，并设计了书本的读书小报，对人物性格和情节发展等进行了梳理Reading Log和读书小报在教学楼一楼走廊进行了公开展示。
在学科活动的当天，首先，侯宗胜书记为本次活动致辞。他向同学们分享了自己在英语学习中的体会，并提出了英语学习在同学们今后的生活中的重要性，他期望同学们在实际生活学习中提高个人的英语口语表达及书面表达能力，为语言学习打好基础。最后，他期待同学们的精彩表现，并祝愿本次活动能够圆满成功。
书记发言 3
首先出场的《The First School Life》的演员们让所有同学们眼前一亮，他们独特的造型设计和舞台效果让我们跳出了现实，置身于Auggie初到学校时的场景，场下的学生们也感受到了小男孩当时的紧张和手足无措。
《The Performance Space》小组的同学们配合默契，用流利的英语将电影中的情节表现了出来，通过一样情感充沛的发音，讲故事的情节一步步推向高潮。
《They start arguing》的演员们通过丰富的表情、生动的动作和富有感染力的声音，将电影的片段搬到了舞台上，栩栩如生地呈现在了观众们面前。
《First impression》和《The First School Life》不谋而合的选择了相同的片段，他们的倾情演绎让观众们沉浸其中，现场不时掌声雷动，充满了欢声笑语。
《Alien》的演员们生动地演绎了电影中的主人公小男孩Auggie在树林中被校园霸凌的场景，她们的台词功底扎实，动作表情自然流畅，通过细腻的表演和真挚的感情，让观众们仿佛置身到了电影之中。
《Unexpected friendship》的演员们同样也是向大家带来了电影片段中School Bullying的环节，他们情感充沛，表演大方得体，尤其是每一种表情、每一个动作，都发挥得淋漓尽致，尽显人物的内心和品质，将现场的氛围推向了高潮。
最后，八一班的陈怡雯同学带了电影的主题曲《Brand New Eyes》，让我们感受到了Auggie虽然身有“缺陷”，但是因爱与善良、接纳与共情，所谓的“灾难”成就了生命的奇迹，我们要学会接纳生活中的不如意，学会与这个世界和解，那样的话，“每个人，都是一个奇迹，你是，我也是。”
在表演结束后，巢晖校长对本次活动进行了总结发言。他赞扬了同学们惟妙惟肖的表演以及热情洋溢的现场，本次活动激发了同学们学习英语的兴趣，丰富了同学们的校园文化生活，锻炼了同学们大胆说英语能力。他提出，在英语学习中，光光是纸笔上的训练是不够的，同学们更要注重听和说的练习，来达到听说读写的全面发展。最后，他祝愿所有八年级的同学在英语学习上能够取得越来越大的进步。
经过激烈的角逐，最终评选出了最佳服装奖、最佳创意奖、最佳活力奖、最佳人气奖、最佳表演奖和最佳默契奖。
《The First School Life》：最佳服装奖
《The performance Space》：最佳默契奖
《They start arguing》：最佳人气奖
《First impression》：最佳创意奖
《Aliens》：最佳表演奖
《Unexpected friendship》：最佳活力奖
《Brand New Eyes》：最佳创意奖
本次英语学科活动考验学生们的不仅是英语能力，更有对舞美、文学等多学科的融合学习。节目来源于书本，却高于书本。学生们自行修改剧本，增删台词，筹备道具服装，甚至亲手制作道具，用表演赋予文字生命，用感情赋予英语温度，展现了较高的语言交际价值和艺术价值。
